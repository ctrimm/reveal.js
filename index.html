Changes to Installation

Old Way Of Installing Packages

mrt packagename

Nowadays

meteor add author:packagename

Where Packages Live

Used to live in

/packages

Nowadays

.meteor/packages

Functionality

meteor add author:package-name
meteor remove author:package-name
meteor search (even search cached packages offline)

iron:router

Routing specifically designed for Meteor. A router that works on the server and the browser, designed specifically for Meteor.

accounts-password

Meteor core package for managing users

aldeed:collection2

A Meteor package that extends Mongo.Collection to provide support for specifying a schema and then validating against that schema when inserting and updating.

velocity:core

A reactive test-runner for Meteor. Integrates easily with Jasmine, Cucumber, Mocha, CasperJs and Nightwatch
